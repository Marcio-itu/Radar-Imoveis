Os resultados vêm do arquivo imoveis.json, gerado automaticamente pelo GitHub Actions a cada 6 horas. Para adicionar ou remover sites da busca, edite o arquivo fontes.json no repositório. Para forçar uma atualização imediata, vá na aba Actions do repositório → Buscar imóveis → Run workflow.
